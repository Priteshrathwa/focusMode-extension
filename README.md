## ğŸ› ï¸ Usage

1. Click the **Focus Mode** icon in the Chrome toolbar
2. Enter a timer duration (in minutes)
3. Add websites to block (e.g. `youtube.com`, `twitter.com`)

6. You can stop the session anytime
